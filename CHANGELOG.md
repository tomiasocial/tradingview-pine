# Changelog

All notable changes to this project will be documented in this file.

## [v1.0.0] — 2026-05-07
### Added
- `00_full_suite.pine` — all-in-one indicator with master toggles and grouped settings panel
- Full alert suite across all three modules in the combined script

## [v0.3.0] — 2026-05-07
### Added
- `03_auto_trendlines.pine` — auto support/resistance trendlines from pivot swings
- Supply/demand zone boxes at pivot candle bodies (amber fill)
- Touch alerts for support and resistance lines

## [v0.2.0] — 2026-05-07
### Added
- `02_sideway_zone.pine` — sideway range detection with dynamic boxes
- Breakout/breakdown detection with confirmation bars
- Green triangle / red triangle arrow labels on confirmed breakouts

## [v0.1.0] — 2026-05-07
### Added
- `01_trend_candles.pine` — Supertrend-based candle coloring
- Confirmation bar logic to avoid whipsaws
- Alerts for trend direction changes
- Optional Supertrend baseline display
